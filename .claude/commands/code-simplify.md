description: 简化代码以提高清晰度和可维护性——在不改变行为的前提下降低复杂度
---

调用 my-agent-skills:code-simplification 技能。

在保持行为完全一致的前提下，简化最近更改的代码（或指定范围）：

1. 阅读 CLAUDE.md 并研究项目规范
2. 确定目标代码——除非指定了更广的范围，否则聚焦于最近的变更
3. 在修改之前，理解代码的目的、调用者、边界情况以及测试覆盖情况
4. 扫描简化机会：
   - 深层嵌套 → 使用守卫子句（guard clauses）或提取辅助函数
   - 过长函数 → 按职责拆分
   - 嵌套三元运算符 → 改为 if/else 或 switch
   - 通用命名 → 改为描述性命名
   - 重复逻辑 → 提取为共享函数
   - 死代码 → 确认后移除
5. 逐步应用每项简化——每次更改后运行测试
6. 验证所有测试通过、构建成功，且 diff 干净清晰

如果简化后测试失败，请回滚该更改并重新考虑。使用 `code-review-and-quality` 对结果进行审查。
